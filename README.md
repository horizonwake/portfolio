# Vue 3 Portfolio

A portfolio application built with Vue 3 and Vite. Designed for easy customization and rebranding.

## Project Structure

```
src/
├── components/
│   ├── common/          # Reusable UI components (Button, Card, etc.)
│   ├── home/            # Home page components
│   ├── projects/        # Project page components
│   ├── layout/          # Layout components (Header, etc.)
│   └── icons/           # Icon components
├── views/               # Page components
├── config/              # Configuration files
├── router/              # Vue Router setup
└── styles/              # Global styles
```


## Installation

```bash
npm install
```

## Development

```bash
npm run dev
```

## Build for Production

```bash
npm run build
```

## Adding Images

Place images in the `public/images` folder:
- Profile photo: `public/images/profile.jpg`
- Project images: `public/images/projects/project-name.jpg`

## Technologies Used

- Vue 3
- Vue Router
- Vite
- CSS Variables for theming
